# Camel Dojo

A collection of hands-on examples for learning Apache Camel integration patterns with various enterprise technologies.

## Project Structure

| Directory | Description |
|-----------|-------------|
| [security/](security/) | Security-related examples including authentication, authorization, and identity management |

## About

This repository provides practical, runnable examples that demonstrate how to use Apache Camel in real-world enterprise scenarios. Each subdirectory focuses on integration with a specific technology and includes multiple examples covering different use cases.

## Getting Started

Navigate to a subdirectory of interest and follow its README for setup instructions and prerequisites.
